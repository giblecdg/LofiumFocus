# 🎯 LofiumFocus

A minimalist Pomodoro timer with lofi music and ambient sounds.

<img src="https://github.com/user-attachments/assets/0a289424-a33d-4079-b239-26d1bb5ca4c3" />

## ✨ Features

- ⏱️ **Smart Breaks** - Short/long breaks with customizable durations
- 📊 **Progress Ring** - Visual timer with animated progress
- 🔥 **Streak System** - Track your daily focus streak
- 🎵 **Lofi Music** - Built-in playlist with shuffle
- 🌧️ **Ambient Sounds** - Rain, fireplace, white noise, cafe
- 📈 **Weekly Summary** - Statistics with bar chart
- 🐱 **Cute Cat Mascot** - Reacts to your streak!
